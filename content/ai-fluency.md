# AI Fluency & Technical Skills

I use AI as an accelerant for product work — not a replacement for design judgment. Below is a concise summary of tools, practices, and certifications that inform my AI-enabled workflows.

## Tools & Platforms

- Anthropic Claude (Sonnet): primary reasoning and synthesis engine.
- Claude Code: used to prototype Claude-driven features and build parts of my portfolio site.
- Cursor: AI-assisted code editing and iteration.
- Whisper Flow: voice-to-text where needed for research capture.
- Perplexity / Fire Crawl: research and scraping agents for background and competitive analysis.
- GPT models: ideation, edge-case analysis, and microcopy drafts.
- Playwright: end-to-end browser testing for regressions and QA.

## Engineering & Infra

- React + Vite for rapid frontend builds
- Express proxy for secure API access
- Supabase for persistence and auth
- Vercel for deployment

## Practices

- Prompt engineering as a UX decision: intake questions and prompt structure determine output quality.
- Modular prompt phases: extraction → synthesis → prioritization → formatting to make A/B testing and debugging tractable.
- Use AI to draft artifacts (specs, taxonomies, copy) and always validate with users or engineers before shipping.

- Anthropic Skilljar: Claude 101
- Anthropic Skilljar: Claude Code in Action
- Anthropic Skilljar: Introduction to Agent Skills
